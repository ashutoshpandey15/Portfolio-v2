# Ashutosh Pandey's Personal Portfolio

Welcome to the repository for Ashutosh Pandey's personal portfolio website! This site showcases my skills, projects, and experience as a Software Developer with expertise in AI and DevOps.

## 🚀 Getting Started

### Prerequisites
This project is a static website and does not require any specific backend or server-side technologies to run locally. You only need a web browser.

### Installation
To get a local copy up and running, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/portfolio-main.git
   ```
   (Replace `your-username` with the actual GitHub username if this project is hosted on GitHub, or provide the correct repository URL).

2. **Navigate to the project directory:**
   ```bash
   cd portfolio-main
   ```

### Usage
Open the `index.html` file in your preferred web browser to view the portfolio.

```bash
start index.html # On Windows
open index.html  # On macOS
x-www-browser index.html # On Linux (or your default browser command)
```

## ✨ Features

- **Clean and Modern Design**: A minimalist and responsive design that works across all devices.
- **About Me Section**: A brief introduction.
- **Dynamic Text Animation**: Engaging animated text for the hero section.
- **Work Experience & Education Timeline**: Detailed timeline of professional and academic journey.
- **Skills Section**: Showcasing technical proficiencies.
- **Project Showcase**: Highlighted personal and professional projects.
- **Contact Form**: Easy way for visitors to get in touch, powered by Web3Forms for reliable email delivery.
- **Dark/Light Theme Toggle**: User-friendly theme switching for better accessibility.

## 🛠 Technologies Used

- **HTML5**: For structuring the content.
- **CSS3**: For styling and layout, including responsive design.
- **JavaScript**: For interactive elements and dynamic content.
